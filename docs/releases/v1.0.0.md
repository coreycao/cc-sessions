# CC Sessions v1.0.0

This is the first public release of CC Sessions, a desktop app for browsing, searching, organizing, and resuming local Claude Code and Codex CLI sessions.

## Highlights

- Browse Claude Code and Codex CLI session history in a native desktop app.
- Read full conversations with Markdown, syntax highlighting, collapsible tool calls, and session metadata.
- Search across session metadata and indexed conversation content.
- Organize sessions with status, tags, notes, starring, saved messages, and archived-session bulk actions.
- Resume supported sessions through the matching CLI.
- Install future signed updates from inside the app.
- See visible update-check status and retry guidance if GitHub release metadata is slow or unreachable.
- Download updates without immediately restarting; restart only when you are ready.
- Install native macOS builds for Intel and Apple Silicon.

## Installation Notes

This first release must be installed manually from the macOS Intel or Apple Silicon assets below. After installing v1.0.0, future releases can be tested through the in-app updater.

Linux and Windows builds are not published yet because platform-specific app behavior still needs adaptation and validation.
