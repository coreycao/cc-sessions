# CC Sessions v1.0.11

This release focuses on project management shortcuts and a cleaner batch AI rename flow.

## Highlights

- **Open projects from Settings**: project rows now include an open menu for Finder, Terminal, VS Code, and Xcode.
- **Cleaner batch AI rename**: skipped sessions are hidden by default, and a status filter lets you switch between active, all, pending, ready, skipped, failed, and applied sessions.
- **Less sidebar noise**: the compact Stats panel has been removed from the left sidebar. The full Statistics page remains available in Settings.
- **AI status polish**: generation and review loading states are less redundant and easier to scan.
- **Documentation refresh**: README now reflects the current AI, project management, data, updater, and language features.

## Notes

- Project open actions validate that the target path belongs to an indexed local session before launching external apps.
- Batch AI rename still stores generated titles only in CC Sessions metadata and does not rewrite Claude Code or Codex session files.
