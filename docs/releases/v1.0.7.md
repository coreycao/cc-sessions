# CC Sessions v1.0.7

This release adds project management — archive quiet projects, rename them for clarity, and add local notes — so the project list stays focused, all without touching source session files.

## Highlights

- Added a **Manage Projects** settings page to archive and restore projects, edit display names, add local notes, and choose a project icon.
- Added project **sorting controls** in the header: Recent, Active, and Top.
- Added a **project icon picker** with a built-in set of icons; the selected icon is shown throughout the app.
- Project metadata (archive state, display name, note, and icon) is stored locally in CC Sessions app data — original Claude Code and Codex session files are never modified.
- Archived projects can be filtered and restored at any time, and the sidebar respects the archive state.
- Added English and Simplified Chinese translations for every new project-management string.

## Notes

- All project-management data lives in CC Sessions app data alongside existing GTD metadata.
- Archiving a project only hides it from the active project list; the underlying sessions are untouched and the project can be restored.
- Display names, notes, and icons are local-only and do not rename or edit any source session files.
