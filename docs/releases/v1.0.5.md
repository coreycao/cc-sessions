# CC Sessions v1.0.5

This release improves Settings, local data visibility, sidebar polish, and AI provider setup.

## Highlights

- Added a Statistics settings page with project, session, provider, activity, and long-session summaries.
- Added a Data settings page that shows CC Sessions app-data storage usage and includes a manual Sync sessions action.
- Improved AI Provider cards with a calm read-only state and an explicit Edit mode.
- Fixed AI Provider connection tests for OpenAI-compatible APIs that return a valid response without plain string message content.
- Added smooth expand and collapse animations to sidebar Stats and Tags sections.
- Reworked the session-update refresh tooltip so it appears quickly and is no longer clipped by the session list panel.

## Notes

- The Data page reports CC Sessions app data only, such as cache, search index, saved messages, tags, notes, and AI settings.
- Manual Sync sessions reuses the existing global refresh flow and refreshes storage usage afterward.
- Session review generation still requires non-empty AI message content; only the connection test is more permissive.
