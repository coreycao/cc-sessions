# CC Sessions v1.0.4

This release refines update entry points and sidebar statistics.

## Highlights

- Added a title-bar update action that appears only when an app update is available, downloading, or ready to restart.
- Moved session file update refresh from the full-width banner into the session list header.
- Kept the session refresh action discoverable with a hover tooltip while reducing visual noise.
- Expanded sidebar statistics by default.
- Added separate Claude Code and Codex session counts to the sidebar statistics panel.
- Adjusted the statistics collapse arrow so expanded state points downward.

## Notes

- The app update button stays hidden when no app update is available.
- Session file update refresh still uses the existing global refresh loading overlay.
