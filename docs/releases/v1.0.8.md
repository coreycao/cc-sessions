# CC Sessions v1.0.8

This release reworks saved messages — select several at once and save them as a single bookmark, or export your entire collection to Markdown — plus a softer, rounded checkbox throughout.

## Highlights

- **Multi-select saving in the DetailPanel**: a new *Select* toggle in the header (plus **Cmd/Ctrl-click** and **Shift-click** for range selection) lets you pick multiple conversation turns and save them together. A floating bar shows the count with *Select all*, *Save*, and *Cancel* (Esc also exits).
- **Merged bookmarks**: saving two or more turns combines their content into one saved entry instead of separate bookmarks. Each merged entry shows a count badge in the list and a "N messages" label in the detail; the underlying `saved-messages.json` gains a `messageCount` field (old files load unchanged).
- **Export all saved messages**: a new block in *Settings → Data* writes every bookmark into a single Markdown document via the system save dialog. The button shows a spinner only while the content is being assembled.
- **Rounded checkboxes**: the batch-select checkbox in the session list and the new message-selection checkbox now share a softer, accent-filled control that matches the app's rounded style.
- Added English and Simplified Chinese translations for every new string.

## Notes

- Merged bookmarks keep the original messages intact — only the saved entry combines them, and the source session files are never modified.
- Saving a single turn still creates a normal, individual bookmark.
- The export file is plain Markdown, grouped by saved time, and is fully portable.
