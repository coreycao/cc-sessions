# CC Sessions v1.0.1

This maintenance release validates the signed updater flow from v1.0.0 to v1.0.1.

## Changes

- Enabled updater artifact generation for signed release metadata.
- Improved release workflow reliability by using Tauri's native `latest.json` generation.
- Limited release packaging to macOS Intel and Apple Silicon while Linux and Windows support remains under adaptation.
- Stabilized CI tests for the current macOS release path.

## Update Test

Install v1.0.0 first, then use the in-app update check to upgrade to v1.0.1.
