# CC Sessions v1.0.1

This release improves the AI settings workflow and polishes project selection UX.

## Highlights

- Replaced the native AI profile `select` with a project-switcher-style popover for a more consistent UI.
- Added a project-provider badge in the title-bar project switcher, so Claude Code and Codex sources are easier to distinguish.
- Shortened project labels in the project menu and moved full paths into hover tooltips.
- Prevented `Cmd+A` from hijacking editable fields in the AI settings panel.

## Notes

- API keys continue to be stored locally in the app data directory.
- Update and session review flows remain unchanged in behavior; this release focuses on UX and keyboard handling.
