# CC Sessions v1.0.2

This release adds bilingual UI support for English and Chinese.

## Highlights

- Added a lightweight i18n layer with English and Chinese dictionaries.
- Added a language selector in Settings > Appearance.
- Persisted the selected language locally so it is restored after restart.
- Localized the main app shell, sidebar, project switcher, settings pages, AI settings, session list, session detail actions, saved messages, common dialogs, and toast feedback.
- Updated component tests to render inside the i18n provider.

## Notes

- The default display language is English.
- User-generated content, project names, tags, API profile names, and exported Markdown content are not translated.
