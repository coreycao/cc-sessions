# CC Sessions v1.0.9

This release adds AI-assisted session tagging and cleans up the DetailPanel action layout so session tools feel quieter and easier to scan.

## Highlights

- **AI tag suggestions**: generate concise tag suggestions for the current session using your configured OpenAI-compatible provider. Suggestions can be added one by one or all at once, and existing tags are filtered out.
- **Robust tag parsing**: AI tag responses are normalized on the Rust side and tolerate JSON arrays, Markdown-wrapped JSON, and simple comma or newline text.
- **Cleaner DetailPanel actions**: Review, Rename, AI tags, and Star now live together in a compact metadata toolbar. The title bar stays focused on close, title, archive, and overflow actions.
- **Simpler overflow menu**: duplicate Review placement was removed; secondary actions like resume, selection mode, display mode, export, and delete remain in More.
- Added English and Simplified Chinese translations for the new toolbar and tag-generation strings.

## Notes

- AI-generated tags are stored only in CC Sessions metadata. Source Codex and Claude Code session files are not modified.
- Tag generation uses a compact session context to keep requests smaller than full-session review prompts.
