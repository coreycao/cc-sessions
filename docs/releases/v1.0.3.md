# CC Sessions v1.0.3

This release improves responsiveness when browsing and refreshing sessions.

## Highlights

- Virtualized the session detail conversation with TanStack Virtual.
- Virtualized the session list with the same library for a unified scrolling implementation.
- Improved long conversation switching with worker-based parsing and measured dynamic row heights.
- Made session row selection feel immediate by painting the selected state before loading detail content.
- Added a blocking global loading overlay while manually refreshing session updates.
- Improved refresh and scanner error handling so failed scans no longer look like an empty session list.

## Notes

- The conversation renderer still truncates very long individual messages inline and keeps the full content available through the expand action.
- This release focuses on macOS app responsiveness and does not add Linux or Windows packaging.
